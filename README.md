🛒 SmartCart Customer Segmentation System
📌 Project Summary

Developed an end-to-end Customer Segmentation System using Unsupervised Machine Learning to identify distinct customer groups for a growing e-commerce platform. The system analyzes customer demographics, spending behavior, purchase frequency, and engagement patterns to enable data-driven marketing strategies.

🎯 Problem Statement

SmartCart was using generic marketing strategies for all customers, leading to inefficient campaigns and missed high-value opportunities. The objective was to segment customers into meaningful clusters based on behavioral patterns.

⚙️ What I Implemented

✔️ Data Cleaning & Missing Value Handling

✔️ Feature Engineering (Age, Customer Tenure, Total Spending, Total Children)

✔️ Outlier Detection & Removal

✔️ Categorical Encoding using OneHotEncoder

✔️ Feature Scaling using StandardScaler

✔️ Dimensionality Reduction using PCA (3 components)

✔️ KMeans Clustering for customer segmentation

✔️ Optimal Cluster Selection using Elbow Method & Silhouette Score

✔️ 3D Cluster Visualization

🛠️ Tech Stack

Python | Pandas | NumPy | Scikit-learn | PCA | KMeans | Matplotlib | Seaborn

📊 Outcome

Successfully segmented 2240 customers into meaningful clusters, uncovering hidden behavioral patterns. The model enables targeted marketing, improved retention strategies, and better business decision-making.
